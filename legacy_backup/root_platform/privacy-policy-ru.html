Политика конфиденциальности

Обновлено: 5 марта 2025

1. Введение

InfoLady ("мы", "наш", "нас") управляет веб-сайтом infolady.online. Данная Политика конфиденциальности объясняет, как мы собираем, используем и защищаем информацию, которую вы предоставляете при использовании наших услуг.

2. Какую информацию мы собираем

Мы можем собирать следующие типы информации:

  • Информация об аккаунте: Имя и адрес электронной почты при регистрации или входе
  • Данные об использовании: URL-адреса сайтов, которые вы отправляете для анализа, посещенные страницы, проведенное время
  • Платежные данные: Платежи обрабатываются PayPal. Мы не храним данные платежных карт
  • Данные лидов: Адрес электронной почты и URL-адрес сайта, отправленные перед созданием платного отчета
  • Файлы Cookie: См. нашу Политику использования cookie для подробностей

3. Как мы используем вашу информацию

  • Для предоставления SEO-анализа, отчетов аудита и инструментов, которые вы запрашиваете
  • Для отправки вам результатов отчетов и уведомлений, связанных с сервисом
  • Для поддержки и улучшения нашей платформы
  • Для обработки платежей через PayPal
  • Для отправки вам опциональных маркетинговых сообщений (вы можете отказаться в любое время)

4. Хранение данных

История сканирования и данные аккаунта в настоящее время хранятся локально в localStorage вашего браузера. Никакие личные данные не передаются на внешние серверы, за исключением интеграций вебхуков (Google Таблицы, Telegram), используемых для сбора лидов, и PayPal для обработки платежей.

5. Файлы Cookie

Мы используем функциональные файлы cookie для сохранения вашей сессии и настроек. См. нашу Политику использования cookie для получения полной информации.

6. Сторонние сервисы

Мы используем следующие сторонние сервисы, которые имеют собственные политики конфиденциальности:

  • PayPal — для обработки платежей
  • Google Analytics — для анонимной статистики использования
  • Google Fonts — для шрифтов
  • Telegram Bot API — для внутренних уведомлений о лидах

7. Партнерские ссылки

Некоторые ссылки на InfoLady являются партнерскими. Мы можем получать комиссию, когда вы совершаете покупку по этим ссылкам, без дополнительных затрат для вас. Партнерские ссылки помечены rel="nofollow sponsored".

8. Ваши права

9. Конфиденциальность детей

Наши услуги не предназначены для лиц младше 16 лет. Мы сознательно не собираем личные данные детей.
