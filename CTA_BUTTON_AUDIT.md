## 3. Telegram Bot - Параметры стартовых команд ✅

### Обработка параметров
В backend/bot.js функция `parseStart()` распознает:

| Параметр | Язык | Сегмент | Действие |
|----------|------|---------|----------|
| `mobile_cta` | auto (EN/RU по URL) | warm | Приветствие + лид-форма |
| `global_cta` | auto | warm | Приветствие + лид-форма |
| `audit_root_index` | auto | warm | Приветствие |
| `audit_root_ru` | RU | warm | Приветствие (РУ) |
| `audit_root_en` | EN | warm | Приветствие (EN) |
| `paid_ru` | RU | purchased | Специальные сообщения для оплативших |
| `paid_en` | EN | purchased | Специальные сообщения для оплативших |

### Код обработки
```javascript
const knownCtaParams = ['global_cta', 'mobile_cta', 'audit_root_index', 'audit_root_ru', 'audit_root_en', 'paid_ru', 'paid_en'];
if (knownCtaParams.includes(decoded)) {
    const lang = decoded.includes('en') ? 'en' : 'ru';
    const isPaid = decoded.startsWith('paid_');
    return { 
        source: 'site', 
        lang, 
        step: isPaid ? 'paid' : 'chat', 
        segment: isPaid ? 'purchased' : 'warm', 
        domain: '' 
    };
}
```

**Статус:** ✅ Все параметры правильно обрабатываются
