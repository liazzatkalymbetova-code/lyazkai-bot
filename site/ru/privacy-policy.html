Политика конфиденциальности

Обновлено: 5 марта 2025 г.

1. Введение

InfoLady («мы», «наш», «нас») управляет сайтом infolady.online. Данная Политика конфиденциальности объясняет как мы собираем, используем и защищаем информацию, которую вы предоставляете при использовании наших услуг.

2. Информация, которую мы собираем

Мы можем собирать следующие типы информации:

  • Информация об аккаунте: Имя и адрес электронной почты при регистрации или входе в систему
  • Данные об использовании: URL-адреса веб-сайтов, которые вы отправляете на анализ, посещенные страницы, затраченное время.
  • Данные платежа: Платежи обрабатываются PayPal. Мы не храним оплату данные карты
  • Данные лида: Адрес электронной почты и URL-адрес веб-сайта, отправленные до создания платного отчет
  • Файлы cookie: Подробную информацию см. в нашей Политике использования файлов cookie.

3. Как мы используем вашу информацию

  • Для предоставления SEO-анализа, аудиторских отчетов и инструментов, которые вы запрашиваете.
  • Для отправки вам отчетов о результатах и ​​сообщений, связанных с обслуживанием.
  • Для поддержания и улучшения нашей платформы
  • Для обработки платежей через PayPal
  • Для отправки вам дополнительных маркетинговых сообщений (вы можете отказаться в любое время)

4. Хранение данных

История сканирования и данные учетной записи в настоящее время хранятся локально в localStorage вашего браузера. Нет персональные данные передаются на внешние серверы, за исключением интеграции вебхуков (Google Таблицы, Telegram) используется для сбора потенциальных клиентов, а PayPal — для обработки платежей.

5. Файлы cookie

Мы используем функциональные файлы cookie для поддержания вашего сеанса и предпочтений. Посмотрите наш Политика использования файлов cookie для полного детали.

6. Сторонние сервисы

Мы используем следующие сторонние сервисы, которые имеют собственную политику конфиденциальности:

  • PayPal — для обработки платежей
  • Гугл Аналитика — для анонимной статистики использования
  • Google Шрифты — для типографики
  • API Telegram-бота — для внутренних уведомлений лидов

7. Партнерские ссылки

Некоторые ссылки на InfoLady являются партнерскими. Мы можем получать комиссию при покупке через эти ссылки без каких-либо дополнительных затрат с вашей стороны. Партнерские ссылки отмечены значком rel="nofollow sponsored".
